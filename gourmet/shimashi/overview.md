# 志摩市 グルメ調査 - 概要

## 旅行情報

- **訪問期間**: 2026-09-25 〜 2026-10-04（全体行程の一部）
- **宿泊**: 予約済み（詳細は別途確認）
- **主要エリア**: 賢島、的矢湾、英虞湾（あごわん）、浜島
- **主要駅**: 賢島駅（近鉄志摩線終点）

## 志摩市の食の特徴

### 海の幸の宝庫
志摩市は伊勢志摩国立公園内に位置し、的矢湾・英虞湾などリアス式海岸に面した漁業の町。新鮮な海産物が豊富。

### 主要な地元料理・名物

1. **的矢かき（まとやがき）**
   - 志摩市的矢湾の名産ブランド牡蠣
   - 浄化海水で育つため生食も安心、ぷりぷり食感と強い甘味
   - シーズン：11月〜3月（冬季）
   - 調理法：カンカン焼き、生牡蠣、カキフライ、牡蠣づくし定食
   - **注意**: 9月訪問のため、牡蠣ハイシーズン前 → 優先度中〜低
   - **夏季は岩牡蠣**が旬（5月〜8月頃）→ 9月訪問で一部提供可能性あり

2. **伊勢海老（いせえび）**
   - 志摩の高級食材
   - 刺身、姿造り、鬼殻焼き、みそ汁など
   - 旬：秋〜春（9月訪問は秋の始まり、提供店多い）

3. **あのりふぐ**
   - 志摩の地元ブランドふぐ
   - シーズン：冬季限定（10月〜3月）
   - **注意**: 9月訪問のため、シーズン前 → 対象外

4. **手こね寿司（てこねずし）**
   - カツオやマグロの漬けを使った郷土料理
   - 伊勢志摩地域の伝統料理

5. **真珠貝・鮑（あわび）**
   - 志摩は真珠養殖で有名、真珠貝の寿司も提供
   - 鮑は高級食材として刺身や会席料理で

6. **海鮮料理全般**
   - 地元の新鮮な魚介（真鯛、アジ、サバなど）
   - 海鮮丼、刺身盛り合わせ、定食

## 調査戦略

### 訪問時期による優先順位

**9月訪問の特徴**:
- 的矢かき：ハイシーズン前（11月〜3月）だが、岩牡蠣が9月上旬まで可能性あり → 優先度中
- 伊勢海老：秋の始まり、旬の入り口、提供店多い → 高優先度
- あのりふぐ：シーズン前（10月〜3月）→ 対象外
- 手こね寿司：通年提供 → 高優先度
- 海鮮全般：通年提供、秋の魚も美味しい時期 → 高優先度

### 優先順位

1. **伊勢海老料理（高優先度）**
   - 姿造り、会席料理、定食
   - 専門店や和食店
   - 刺身やアオサあんかけなど

2. **海鮮料理全般（高優先度）**
   - 手こね寿司の名店
   - 海鮮丼や刺身定食
   - 地魚料理の評価店

3. **的矢かき専門店（中優先度）**
   - 通年営業の牡蠣専門店（岩牡蠣の提供可能性）
   - 的矢湾周辺の養殖場直営店
   - **注意**: 9月は的矢かきのハイシーズン外のため、店舗によっては休業または限定メニューの可能性

4. **カフェ・デザート（中優先度）**
   - 英虞湾を望むカフェ
   - 地元スイーツ

### リサーチアプローチ

- 伊勢海老料理の店舗を5〜7店舗詳細調査
- 手こね寿司の名店を2〜3店舗調査
- 海鮮料理全般の評価店を3〜5店舗調査
- 的矢かき専門店を2〜3店舗調査（9月営業状況確認）
- カフェ・デザートは2〜3店舗選定

## 重要な制約条件

- **予約**: 人気店や伊勢海老料理は事前予約推奨（特に旅館・高級店）
- **定休日**: 各店舗の休業日を要確認（月曜・火曜休みが多い傾向）
- **営業時間**: ランチ・ディナーの時間帯を事前確認
- **シーズン**: 9月訪問のため、的矢かきはハイシーズン外（岩牡蠣の可能性あり）
- **シーズン**: 9月訪問のため、あのりふぐはシーズン外
- **交通**: 賢島・的矢エリアは電車アクセス可、一部店舗は車必須
